# BrowseNowRIP

## What is this?

これは [BrowseNow](https://github.com/kamaboko123/BrowseNow)のブックマークレットオンリー実装です。  
[Twitter公式のシェア用URL](https://twitter.com/intent/tweet)に依存しています。  
サーバなどは不要です。  

## How to use

下記のページを参照してください。

https://iranika.github.io/BrowseNowRIP/index.html
